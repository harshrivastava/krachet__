Getting Started

How to use KRACHET?

Simply click the microphone button and start speaking, or type your message in the chat interface. KRACHET will respond to your queries in real-time. The system is designed to understand natural language, so you can speak or type as you would normally communicate.

Voice Commands

KRACHET supports various voice commands. Try saying "help" to get a list of available commands. You can also use commands like "clear chat" to start a new conversation or "switch to text" to change input methods.

First-time Setup

When you first use KRACHET, you'll need to grant microphone permissions in your browser. Click "Allow" when prompted to enable voice interaction. You can change these permissions later in your browser settings.

Chat Interface

The chat interface allows for both text and voice interactions, making it easy to communicate with KRACHET. Messages are displayed in a clean, modern format with clear visual distinction between user and AI responses.

KRACHET maintains context throughout your conversation, allowing for more natural and coherent interactions. The system remembers previous exchanges and can reference them in future responses.

Troubleshooting

Microphone Issues

If your microphone isn't working, check your browser permissions and ensure your microphone is properly connected. Try refreshing the page and granting permissions again if needed.

Connection Problems

If you're experiencing connection issues, check your internet connection and try refreshing the page. Ensure you have a stable internet connection for optimal performance.

Response Delays

If you notice delays in responses, it might be due to high server load or network latency. Try clearing your browser cache or switching to a different network if the issue persists.

Voice Recognition Accuracy

If you're experiencing issues with voice recognition accuracy, try speaking clearly and in a quiet environment. The system works best with clear audio input and minimal background noise.
